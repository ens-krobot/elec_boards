long getIsens ( char  axis  ) 

Récupère la mesure du courant dans un moteur.

Cette fonction donne une valeur "instantanée moyenne" du courant dans le moteur. En fait, la valeur renvoyée est déjà une moyenne sur un certain nombre d'échantillons.

Paramètres:
axis l'axe moteur dont on veut le courant, peut valoir :
MOTOR_RIGHT le moteur de droite
MOTOR_LEFT le moteur de gauche
Renvoie:
Isens courant mesuré, en mA (entier sur 16 bits)
